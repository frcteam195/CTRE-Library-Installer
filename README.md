# CTRE Library Installer (Unofficial)

![CTRE logo](https://github.com/CrossTheRoadElec/Phoenix-Documentation/blob/master/images/General-Title.png?raw=true)

A script to automate the installation process of the CTRE Libraries into the wpilib folder

This script will install the CTRE user libraries into your wpilib user folder. This operation will **OVERWRITE** the current CTRE libraries that are installed.
